**_Aradigin Cumle Web_**

## Technologies

- **Html**
- **Css**

## Dependencies

- *Vue-Router*
- *Axios*

## Project Setup Process

### Install npm (for npm packages)

```sh
npm install
```

### to run project in development environment

```sh
npm run dev
```

### Compile and Minify for Production

```sh
npm run build
```

### Lint with

```sh
npm run lint
```
