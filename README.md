# Aradigin-Cumle-Web

***Aradigin Cumle Web Project***

## Technologies
* **Vue**
* **Css**
* **Js**

## Project Setup Process

### * install npm (for npm packages)
```sh
npm install
```
### * to run project in development environment
```sh
npm run dev
```

* Compile and Minify for Production
```sh
npm run build
```

### * Lint with
```sh
npm run lint
```
